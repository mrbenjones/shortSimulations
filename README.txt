spinThrust.py outputs tables of position, orientation, acceleration components
 and error estimates for a scenario of a ship accelerating forward
at a gravities while spinning one every {rot} cycles.  If you run

>python spinThurst.py {input files}


, it will output one tab delimited file for each input file, each output file
using the inputs to produce the table.
